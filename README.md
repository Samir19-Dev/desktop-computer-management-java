# Sistema de Gestión de Computadores 💻

Aplicación de escritorio desarrollada en Java que permite gestionar computadores mediante operaciones CRUD.

## 🚀 Funcionalidades
- Agregar computadores
- Buscar computadores
- Actualizar información
- Eliminar registros
- Ordenar datos (ascendente y descendente)
- Exportar datos a archivo
- Importar datos desde archivo

## 🛠️ Tecnologías
- Java
- Swing

## 📌 Arquitectura
- Interfaz gráfica (Swing)
- Lógica de negocio (MetodosComputador)
- Modelo (computador)

## ▶️ Cómo ejecutar
1. Abrir el proyecto en NetBeans
2. Ejecutar `FrmComputador.java`
